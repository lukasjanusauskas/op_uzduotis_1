- Duomenys saugomi struktūroje `Studentas`
- Galutinis pažymys skaičiuojamas pagal formulę: *0.4 * namų darbų pažymys + 0.6 * egzamino pažymys*.

- Išvedus studentai yra išrikiuoti.
